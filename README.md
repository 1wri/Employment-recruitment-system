## 一、项目介绍

基于SpringBoot的大学生就业招聘系统的设计与实现

【角色与功能】

管理员(Admin)：

个人信息管理、用户管理、企业管理、岗位分类管理、招聘信息管理、岗位申请管理、在线留言管理、求职信息管理、邀请面试管理。这些功能让管理员能够全面掌控系统运作，包括维护用户账户、企业信息，发布与审核岗位信息，处理求职申请，管理交流互动内容，以及安排面试等。

用户(User)

查看招聘信息、发布求职信息、申请岗位、在线留言。用户可以浏览系统上的职位空缺，主动投递个人简历应聘，同时也能主动发布自己的求职意向，并通过留言功能与招聘方沟通。

企业(Enterprise)

发布招聘岗位信息、下载简历、发送面试邀请。企业用户在收到求职者申请后，能够直接从系统下载对方的简历进行评估，对于满意的候选人则可以发起面试邀请，促进招聘流程的高效进行。

【技术架构】

SpringBoot、Mybatis-Plus、MySQL、Vue、ElementUI、Bootstrap、Layui
